5. Выяснил в каком нахожусь каталоге (pwd, ls)
 5.1 Перешел в корень файловой системы (cd /, pwd, cd~,)
 5.2 Перешел в каталог /var/log и вывел сождержимое (cd /var/log, Ls -1)
 5.3 Перешел в tmp по относительному пути (cd../../tmp)
